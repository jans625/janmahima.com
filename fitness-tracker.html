Fitness Tracker

A fitness and habit tracker on the web that helps users maintain consistency through visual streaks, progress, and local data persistence

Developed a fully functional, user-friendly fitness and habit tracker from scratch to help users build and maintain healthy routines through a streak tracker and goal tracker.

Key Features & Contributions

  • Designed and built the entire website from scratch using HTML, CSS, and JavaScript.
  • Implemented habit, workout, and other creation, daily check-in systems, goal tracking systems, and automatic streak counting.
  • Added date-based tracking with a calendar view and historical progress visualisation.
  • Used the browser's local storage to save all user data persistently across sessions.
  • Created an intuitive and responsive user interface with motivational streak counters and progress indicators.
  • Added features such as search, categories, notes, reset and delete options, and clean data management.

Applications/Software/Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Visual Studio Code
  • Local Storage
  • Responsive Design
  • GitHub

Outcome

Successfully created a clean and private habit tracker that works entirely in the web browser without needing accounts or servers. Demonstrated strong front-end development skills by handling state management, data persistence, and dynamic UI updates using JavaScript. Built a practical tool that encourages consistent fitness and habit formation through visual feedback and streak motivation.
